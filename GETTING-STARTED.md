## 🎯 Quick Start (5 Minutes)

### 1. Create GitHub Account
- Go to [github.com/signup](https://github.com/signup)
- Sign up (free)

### 2. Create Repository
- Click **"+"** → **"New repository"**
- Name: `songdrop`
- Visibility: **Public**
- License: **MIT License**
- **Do NOT check** "Initialize with README"
- Click **"Create repository"**

### 3. Upload Files
- Click **"uploading an existing file"**
- Drag and drop all your SongDrop files
- Commit message: `Initial commit: Add SongDrop application`
- Click **"Commit changes"**

### 4. Enable GitHub Pages
- Go to **Settings** → **Pages**
- Source: **Branch: main**, Folder: **/ (root)**
- Click **"Save"**
- Wait 1-2 minutes

### 5. Visit Your Site
- Your app is live at: `https://yourusername.github.io/songdrop/`
- Share the link and start dropping songs! 🎵

## 🐛 Common Issues

### Site shows 404?
- Wait 2-3 minutes after enabling Pages
- Check file is named `index.html` (lowercase)
- Verify repository is Public

### Changes not showing?
- Wait 1 minute for rebuild
- Clear browser cache (Cmd/Ctrl + Shift + R)

### Need more help?
- Check **SETUP.md** troubleshooting section
- Open an issue on GitHub
- Check [docs.github.com/pages](https://docs.github.com/pages)
